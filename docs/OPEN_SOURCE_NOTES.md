# Open Source Notes

This folder is the public-release package for the MT4 balanced panel.

## Intent

The package is meant to stand on its own as an independent implementation of general price-action ideas. It should not be described as:

- an official port of a closed-source indicator
- a private-community release
- a cracked or converted `ex4` or `ex5`
- an affiliated product of any third-party trading group

## Safe Publishing Checklist

- keep the neutral package name
- keep the `LICENSE` file in the repository
- keep the README non-affiliation wording
- avoid uploading screenshots that contain third-party branding
- avoid copy-pasting paid-community marketing text into the repo

## Forking

If you fork this package, keep the documentation honest about what was independently implemented and what remains your own opinion or trading workflow.
